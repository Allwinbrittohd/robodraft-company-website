Desktop Robot - JTH Series

High-precision, fully automated Cartesian robot system for desktop manufacturing processes.

Product Overview: Cartesian Robot Technology

The Toyo **JTH Series Desktop Robot** operates on a **three-axis coordinate system** similar to the Cartesian system ($X, Y, Z$). This structure ensures highly precise and controllable motion, making it ideal for delicate and complex automation tasks.

Each axis is equipped with **linear actuators or motors** that enable highly accurate linear motion, which can be controlled independently by a computerized control system. This ensures the robot moves in straight lines along each axis.

The JTH series is controlled by the **TSC400 controller**, which manages all axis motions based on programmed instructions, offering comprehensive programming and operational control.

Key Features

  • **Fully Ball Screw Driven:** Improves single and multi-axis positioning accuracy.
  • **Z-Axis Ball Screw System:** Enhances overall system accuracy and reduces the chance of missed motor steps.
  • **High-Performance Option (JTHS):** Offers no missed steps, high speed, and high precision for high load applications.
  • **XY Axis Cover:** Prevents unwanted foreign object inclusion.
  • **Integrated Cable Tracks:** Built-in cable management improves both safety and aesthetic appeal.
  • **TSC400 Controller:** Dedicated control and programming system for the Cartesian robot.
  • **LCD Panel Interface:** Provides easy access for recipe selection, emergency stop, and dispense override buttons.

Dispensing Functions

The JTH desktop robot is highly capable in automated fluid dispensing, ensuring precision and uniformity for various applications, including sealing, bonding, and coating.

Applications

Cutting (Machining)

Performs **precise abstract cutting** by placing the PCB on the $XY$ axis moving table and mounting the matching tool on the $Z$ axis single linear actuator for controlled material removal.

Soldering

Equipped with an automatic soldering tool, the robot can achieve **fully automated soldering** by placing the PCB on the sliding table and executing programmed soldering paths.
